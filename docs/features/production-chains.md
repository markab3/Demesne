# Production Chains

**Status:** Draft

Quality propagates up production chains. High-quality inputs worked by low-quality specialists produce mid-tier output. Every step in the chain contributes.

| Good | Inputs | Uses | Notes |
|---|---|---|---|
| Bread | Wheat/Rye + Labor | Citizen food, army supply | Staple — baseline happiness |
| Salted Meat | Meat + Salt | Army supply, trade good, citizen food | Enables long campaigns |
| Salted Fish | Fish + Salt | Citizen food, army supply, trade | Coastal production |
| Preserved Vegetables | Root Veg + Salt | Citizen food, famine buffer | Low prestige, high stability value |
| Beer | Barley + Hops + Labor | Citizen happiness, trade good | Happiness bonus, modest prestige |
| Wine | Grapes + Labor + Barrels | Citizen happiness, luxury trade, church use | High prestige, climate-restricted |
| Spirits | Grain + Labor + Copper Still | Luxury trade, military morale | Higher prestige than beer |
| Olive Oil | Olives + Labor | Cooking, lamp fuel, soap input | Intermediate and finished good |
| Spiced Goods | Food + Spices + Labor | Luxury food, high prestige trade | Spice rarity makes this extremely valuable |
| Sugar | Sugarcane + Labor | Luxury, confectionery input | Raw form already a luxury |
| Confectionery | Sugar + Flour + Labor | Luxury citizen good, high prestige | Requires two rare inputs |

## Textiles & Clothing

| Good | Inputs | Uses | Notes |
|---|---|---|---|
| Raw Wool | Sheep (tile resource) | Intermediate — wool trade | Exported raw historically |
| Washed Wool | Raw Wool + Water + Labor | Intermediate | Cleaning before spinning |
| Yarn / Thread | Washed Wool / Flax + Labor | Intermediate textile input | Low skill requirement |
| Linen Cloth | Flax Yarn + Labor | Intermediate, basic clothing input | Cheaper than wool cloth |
| Wool Cloth | Wool Yarn + Labor + Specialist | Intermediate, clothing input, trade good | The finishing industry specialization |
| Dyed Cloth | Wool/Linen Cloth + Dye + Specialist | Intermediate, luxury clothing input | Dye source is its own supply chain |
| Dye | Woad / Madder + Labor | Intermediate | Woad = blue, Madder = red |
| Basic Clothing | Linen Cloth + Labor | Citizen happiness, army supply | Low prestige, broad utility |
| Fine Clothing | Dyed Wool Cloth + Specialist | Citizen luxury, prestige good, noble gift | High prestige end product |
| Tapestry | Dyed Wool Cloth + Specialist | Prestige building decoration, luxury trade | Pure prestige, no functional use |
| Rope | Flax / Hemp + Labor | Construction input, naval supply | Essential intermediate |
| Sails | Linen Cloth + Rope + Labor | Shipbuilding input | No ships without this chain |

The wool chain is the canonical multi-city example: Raw Wool → Washed Wool → Yarn → Wool Cloth → Dyed Cloth → Fine Clothing (six steps). No single city should find it efficient to run all six steps at high quality. The historical pattern — raw wool from pastoral cities, finishing in specialist ones — should emerge naturally from the quality and specialization system.
